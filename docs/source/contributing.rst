Contributing
============

We welcome contributions to Pantheon!

How to Contribute
-----------------

1. **Fork the Repository**

   Visit https://github.com/aristoteleo/pantheon-agents

2. **Clone and Install**

   .. code-block:: bash

      git clone https://github.com/YOUR_USERNAME/pantheon-agents.git
      cd pantheon-agents
      pip install -e .

3. **Make Your Changes**

   - Write clear, documented code
   - Follow existing patterns
   - Add tests when applicable

4. **Submit a Pull Request**

   - Describe what changes you made
   - Reference any related issues

Areas for Contribution
----------------------

- **New Examples**: Demonstrate agent capabilities
- **Bug Fixes**: Help improve stability
- **Documentation**: Clarify and expand guides
- **New Features**: Extend functionality

Code Style
----------

- Follow PEP 8
- Use type hints where appropriate
- Add docstrings to functions

Testing
-------

Run tests before submitting:

.. code-block:: bash

   pytest

Questions?
----------

Open an issue on GitHub for discussion.
